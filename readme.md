# Jekyll Mix

**A Jekyll Boilerplate With Laravel Mix**

## Features
* ES2017 + modules compilation
* BrowserSync
* Sass
* PostCSS Autoprefixer
* Minification

## Requirements

* node
* Jekyll
* npm or Yarn

## Installation

```yarn```

## Usage

Make sure to edit files in assets/_js and assets/_scss

### Development

```yarn watch```

### Static build

```yarn production```
